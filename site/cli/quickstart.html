介绍 ​

  1. 快速生成基于 WlinkUI 的 脚手架模板，更加快速的启动一个项目的开发。
  2. 快速生成基于 DCLOUD 的移动端项目。不用在为配置给中环境，书写常用 utils ，请求等烦恼，一步到位。直接生成，加速快发。
  3. 生成常用业务组件模板，对比于 plop、code snippets 等生成的模板更加快速以及易用。
  4. 对于 wlink 组件库功能的快速开发，能够做到直接生成 Less 文件、vue 组件文件、同步生成 doc 基本示例。从而能更加便捷的为 wlink 添砖加瓦
  5. 更多功能

安装 ​

npm

bash
npm install -g @wlink/cli

pnpm

bash
pnpm add -g  @wlink/cli

使用 ​

bash
wcli

效果

wcli -v
